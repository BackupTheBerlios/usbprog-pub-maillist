Hallo Liste,

ich habe usbprog erfolgreich zusammengeloetet, allerdings gibt es
enorme Probleme unter Linux (Ubuntu, Kernel 2.6.20).

Das initiale Flashen habe ich unter Windows durchgefuehrt, dies verlief
erfolgreich. Das erste Firmware-Flash via USB unter Windows
(Jumperstellung 1-2) ebenso.

Es ist mir allerdings nicht gelungen, das Interface erfolgreich unter
Linux anzusprechen. Die LED leuchtet gruen.

Um Rechteprobleme zu vermeiden, habe ich folgende Befehle als 'root'
ausgefuehrt.

Ausgabe 1 (Klassischer Flashversuch):
root at dreckskiste:~/AVR/Debug# avrdude -c avrispv2 -P usb -p m8
avrdude: usbdev_open(): did not find any USB device "usb"

Ausgabe 2 (Firmeware-Bestueckungsversuch)
root at dreckskiste:~/Temp# ./usbprog avrispmk2.bin
usbprog GNU/GPL2 Update Tool
Author: Benedikt Sauter, sauter at ixbat.de 2007
Firmware Pool: http://www.embedded-projects.net/usbprog

Error: Can't find vaild usbprog adapter on usb bus.
                        Be sure that you are root or have enough
permissions to access usb.


In der folgenden Adresse habe ich alle, meines Erachtens relevante
Daten, zusammengestellt.

https://www2.hs-esslingen.de/~hoadit00/usbprog

Anscheinend ist das Problem schon bekannt. Ich hoffe ich kann zur
Loesung beitragen.
