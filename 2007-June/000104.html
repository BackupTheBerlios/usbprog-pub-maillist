Hallo Marcel,

so kann man den usbprog Adapter mit der Firmware AVRISPmk2 unter BASCOM
nutzen.

Ich hab gerade das neuste WinAVR Paket heruntergeladen.

Von der Seite:
http://sourceforge.net/project/showfiles.php?group_id=68108, die Datei
WinAVR-20070525-install.exe .

Das installieren, danach sollte man mit dem Konsolenbefehl:

c:\WinAVR-20070525\bin\avrdude.exe -p m32 -c avrispv2 -P usb
die Signatur vom Zielprozessor auslesen können.

Eine .hex Datei kann man dann mit
c:\WinAVR-20070525\bin\avrdude.exe -p m32 -c avrispv2 -P usb -U
flash:w:meinedatei.hex:i in den ATMega32 laden.

Wenn das alles geklappt hat dann ab ins BASCOM, dort bei Programmer den
externen Programmer angeben. Als Programm
c:\WinAVR-20070525\bin\avrdude.exe

und als Parameter:
c:\WinAVR-20070525\bin\avrdude.exe -p m32 -c avrispv2 -P usb -U
flash:w:"{file}":i

Und den Haken .hex nicht vergessen!

So klappt das hier.

Gruss Bene
