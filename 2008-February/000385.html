Hello,

I'm proud to announce version 0.1.4 of USBprog. This release contains
following changes compared to the previous version 0.1.3:

  - Windows: honor IE proxy settings
  - CLI: make (Internet) download progress working
  - CLI: always re-discover devices with the "devices" command
  - CLI: new "start" command to only start the firmware without uploading
    anything and added "-nostart" option to "upload" command
  - Unix: build system fixes


Download it from following sources:

  - Source code for Linux, *BSD and others (all architectures):
    http://prdownload.berlios.de/usbprog/usbprog-0.1.4.tar.bz2
                                                               
  - RPMs for openSUSE and Fedora (i386 and x86-64):            
    http://download.opensuse.org/repositories/electronics/
                                                          
  - Installer for Windows (works on x86 and x64):         
    http://prdownload.berlios.de/usbprog/USBprog-0.1.4.exe

Have a lot of fun and report bugs to this mailing list or the USBprog
forum (http://forum.embedded-projects.net/).
