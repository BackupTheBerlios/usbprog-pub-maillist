Hi,

I'm happy to announce that starting today usbprog is available in the
Debian unstable archives (and thus presumably in Ubuntu very soon). I have
created four binary packages so far out of the 0.1.4 tarball:

libusbprog-dev - Development files for libusbprog
libusbprog0 - Library for programming the USBprog hardware
usbprog - Firmware programming tool for the USBprog hardware
usbprog-gui - GUI firmware programming tool for the USBprog hardware

http://bugs.debian.org/cgi-bin/pkgreport.cgi?which=pkg&data=usbprog&archive=no&version=&dist=unstable

Please report any bugs and suggestions for improvements.
