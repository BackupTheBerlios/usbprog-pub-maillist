Hi,

here's some suggestions for improving usbprog:

 - usbprog-gui should have a --version and --help option. Yes, it's a
   GUI, but many people are used to at least get some version output
   from any binary, whether GUI or not. I think this would be a good
   thing.

 - Can the readline support be enhanced to be more clever? E.g. if you
   use TAB completion it would be good if it would only complete valid
   items.

   Examples:

    - Type "dow", press TAB and it completes to "download".

    - Type "download b", press TAB and it completes to "download
      binkdemo", as that's the only firmware which starts with "b".

      Currently TAB tries to autocomplete filenames in the current
      directory, which is useless here.
