Hello,

I'm proud to announce version 0.1.2 of USBprog. This release contains
following changes compared to the previous version 0.1.1:

  - GUI: new firmware and pin information
  - CLI: new string tokenizer to be able to deal with spaces
    (use "string with spaces" or string\ with\ spaces like in bash)
  - improved error handling
  - autoconf fixes (building without GUI is now possible)


Download it from following sources:

  - Source code for Linux, *BSD and others (all architectures):
    http://www.ixbat.de/usbprog/flashtool/usbprog-0.1.2.tar.bz2

  - RPMs for openSUSE and Fedora (i386 and x86-64):
    http://download.opensuse.org/repositories/electronics/

  - Installer for Windows (works on x86 and x64):
    http://www.ixbat.de/usbprog/flashtool/USBprog-0.1.2.exe

Have a lot of fun and report bugs to this mailing list or the USBprog
forum (http://forum.embedded-projects.net/).
