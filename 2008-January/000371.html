Hello,

I'm proud to announce version 0.1.3 of USBprog. This release contains
following changes compared to the previous version 0.1.2:

  - GUI: fix local upload
  - GUI: make offline mode working, show only firmwares that are
    available offline


Download it from following sources:

  - Source code for Linux, *BSD and others (all architectures):
    http://www.ixbat.de/usbprog/flashtool/usbprog-0.1.3.tar.bz2
                                                               
  - RPMs for openSUSE and Fedora (i386 and x86-64):            
    http://download.opensuse.org/repositories/electronics/
                                                          
  - Installer for Windows (works on x86 and x64):         
    http://www.ixbat.de/usbprog/flashtool/USBprog-0.1.3.exe

Have a lot of fun and report bugs to this mailing list or the USBprog
forum (http://forum.embedded-projects.net/).


       Bernhard
