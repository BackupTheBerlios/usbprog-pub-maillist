Hello,

I'm proud to announce version 0.1.7 of USBprog. This release contains
following changes compared to the previous version 0.1.6:

  - fix problem that firmware with more than 16k can't be flashed
  - add warning to installer that Microsoft Windows Vista is not
    supported

Download it from following sources:

  - Source code for Linux, *BSD and others (all architectures):
    http://prdownload.berlios.de/usbprog/usbprog-0.1.7.tar.bz2
                                                               
  - RPMs for openSUSE, SUSE LINUX Enterprise 10, Fedora and 
    CentOS and RedHat Enterprise Linux (i386 and x86-64):
    http://download.opensuse.org/repositories/electronics/
                                                          
  - Installer for Windows (works on x86 and x64):         
    http://prdownload.berlios.de/usbprog/USBprog-0.1.7.exe

Have a lot of fun and report bugs to the bug tracker
(http://www.embedded-projects.net/bugs/).


       Bernhard
