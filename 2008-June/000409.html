Why do you think that is necessary? You can specify local firmware
files in the GUI. After you are finished, that firmware should be added
to the official pool so that others can download it there.

For your local development, using the command line interface would be
easier anyway. Just specify the firmware file here, for example:

  (usbprog) upload /home/bwalle/projects/usbprog/firmware.bin

instead of

  (usbprog) upload blinkdemo

That's it.
