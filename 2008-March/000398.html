Hello,

I'm proud to announce version 0.1.5 of USBprog. This release contains
following changes compared to the previous version 0.1.4:

  - Linux: add desktop file and install icon
  - GUI: implemented -h / -v / -D command line options for GUI
  - CLI: implement command completion

Download it from following sources:

  - Source code for Linux, *BSD and others (all architectures):
    http://prdownload.berlios.de/usbprog/usbprog-0.1.5.tar.bz2
                                                               
  - RPMs for openSUSE, SUSE LINUX Enterprise 10, Fedora and 
    CentOS and RedHat Enterprise Linux (i386 and x86-64):
    http://download.opensuse.org/repositories/electronics/
                                                          
  - Installer for Windows (works on x86 and x64):         
    http://prdownload.berlios.de/usbprog/USBprog-0.1.5.exe

Have a lot of fun and report bugs to this mailing list or the USBprog
forum (http://forum.embedded-projects.net/).


       Bernhard
