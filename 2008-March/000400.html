Hello,

I'm proud to announce version 0.1.6 of USBprog. This release contains
following changes compared to the previous version 0.1.5:

  - CLI: fix mechanism to avoid downloading the index file again and
    again if the CLI is invoked repeatedly in small intervals
  - remove GPL-incompatible md5_loc.h from source distribution

Download it from following sources:

  - Source code for Linux, *BSD and others (all architectures):
    http://prdownload.berlios.de/usbprog/usbprog-0.1.6.tar.bz2
                                                               
  - RPMs for openSUSE, SUSE LINUX Enterprise 10, Fedora and 
    CentOS and RedHat Enterprise Linux (i386 and x86-64):
    http://download.opensuse.org/repositories/electronics/
                                                          
  - Installer for Windows (works on x86 and x64):         
    http://prdownload.berlios.de/usbprog/USBprog-0.1.6.exe

Have a lot of fun and report bugs to this mailing list or the USBprog
forum (http://forum.embedded-projects.net/).


       Bernhard
