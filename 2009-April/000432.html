Hello,

I'm proud to announce version 0.1.9 of USBprog. This release contains 
following
changes compared to the previous version 0.1.8:

  * Print error when downloading fails with a non 400 status code.
  * Fix GUI on MacOS X (contributed by Nabil Sayegh)

Download it from following sources:

  - Source code for Linux, *BSD and others (all architectures):
    http://prdownload.berlios.de/usbprog/usbprog-0.1.9.tar.gz

    This site is currently offline, therefore with fallback:
    http://bwalle.de/programme/usbprog/usbprog-0.1.9.tar.bz2

  - RPMs for openSUSE, SUSE LINUX Enterprise 10, Fedora and 
    CentOS and RedHat Enterprise Linux (i386 and x86-64):
    http://download.opensuse.org/repositories/electronics/

  - Installer for Windows (works on x86 and x64):
    http://prdownload.berlios.de/usbprog/USBprog-0.1.9.exe

    This site is currently offline, therefore with fallback:
    http://bwalle.de/programme/usbprog/USBprog-0.1.9.exe

Have a lot of fun and report bugs to the bug tracker
(http://www.embedded-projects.net/bugs/).
