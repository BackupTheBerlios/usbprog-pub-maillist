Hello,

I'm proud to announce version 0.1.8 of USBprog. This release contains
following changes compared to the previous version 0.1.7:

  - Fix tab completion for multiple matches (like 'device' and 'devices')
  * Fix problem with recent wxGtk versions that the whole GUI was
    hanging when updating the progress bar.

Download it from following sources:

  - Source code for Linux, *BSD and others (all architectures):
    http://prdownload.berlios.de/usbprog/usbprog-0.1.8.tar.gz

  - RPMs for openSUSE, SUSE LINUX Enterprise 10, Fedora and 
    CentOS and RedHat Enterprise Linux (i386 and x86-64):
    http://download.opensuse.org/repositories/electronics/

  - Installer for Windows (works on x86 and x64): ? ? ? ? 
    http://prdownload.berlios.de/usbprog/USBprog-0.1.8.exe

Have a lot of fun and report bugs to the bug tracker
(http://www.embedded-projects.net/bugs/).


       Bernhard
