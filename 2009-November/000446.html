Hello,

I'm proud to announce version 0.1.10 of USBprog. This release contains
following changes compared to the previous version 0.1.9:

  * Update libusb on Win32 to 0.1.12.2.

(No changes for the other platforms apart from copyright updates.)

Download it from following sources (because Berlios is sometimes not
available, the download is also available on "bwalle.de"):

  - Source code for Linux, *BSD and others (all architectures):
    http://prdownload.berlios.de/usbprog/usbprog-0.1.10.tar.gz
    http://www.bwalle.de/programme/usbprog/usbprog-0.1.10.tar.bz2

  - RPMs for openSUSE, SUSE LINUX Enterprise 10, Fedora and
    CentOS and RedHat Enterprise Linux (i386 and x86-64):
    http://download.opensuse.org/repositories/electronics/

  - Installer for Windows (works on x86 and x64):
    http://prdownload.berlios.de/usbprog/USBprog-0.1.10.exe
    http://www.bwalle.de/programme/usbprog/USBprog-0.1.10.exe

Have a lot of fun and report bugs to the bug tracker
(http://www.embedded-projects.net/bugs/).


       Bernhard
