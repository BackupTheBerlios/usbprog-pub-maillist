Hello,

I'm proud to announce version 0.2.0 of USBprog. This release contains
following changes compared to the previous version 0.1.10:

  * Install PDF documentation and add a new menu entry in "Help"
    that opens a PDF viewer and shows that PDF documentation.
  * Rename "JP" to "JP4" in the pin information dialog to make it more
    clear that the "JP4" jumper is the firmware-specific one.
  * MacOS:
    o Fix the problem that the menu bar is not displayed on MacOS.
    o Build system fixes for MacOS.
  * Windows:
    o Don't install the filter driver on Win32. This makes USBprog
      compatible with Windows Vista and Windows 7.
    o Register the uninstaller on Windows in the Control Panel.
    o Install the driver files in c:\Program Files\USBprog\driver.
    o Ensure that the complete program directory gets uninstalled.

Download it from following sources:

  - Source code for Linux, *BSD and others (all architectures):
    http://prdownload.berlios.de/usbprog/usbprog-0.2.0.tar.gz

  - RPMs for openSUSE, SUSE LINUX Enterprise 10, Fedora and
    CentOS and RedHat Enterprise Linux (i386 and x86-64):
    http://download.opensuse.org/repositories/electronics/

  - Installer for Windows (works on x86 and x64):
    http://prdownload.berlios.de/usbprog/USBprog-0.2.0.exe

The (new!) documentation can also be downloaded as a separate PDF file
at http://prdownload.berlios.de/usbprog/USBprog.pdf.



Have a lot of fun!


Regards,
Bernhard
